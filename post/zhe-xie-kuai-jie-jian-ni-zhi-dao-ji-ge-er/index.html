这些快捷键你知道几个？（二）

2019-10-28 Views 快捷键 212字 1 min read 繁

这篇我们讲讲虚拟桌面相关的快捷键，还有其他几个快捷键。

虚拟桌面

Ctrl + Win + D

按下后新建一个虚拟桌面。

Ctrl + Win + ←/→

这两组快捷键可以对虚拟桌面进行左右切换。

Ctrl + Win + F4

删除虚拟桌面，其中的应用程序和文档会转到附近的桌面中。

其他

Ctrl + Shift + Esc

这个快捷键可以直接打开任务管理器。

Shift + Delete

这个快捷键可以直接删除文件并且删除的文件不会出现在回收站里，所以删除时要三思~

F2

选中文件后按 F2 键可以直接重命名，非常好用。
